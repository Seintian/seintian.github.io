Python è un linguaggio di programmazione molto famoso che, grazie alla sua sintassi particolarmente intuitiva, può essere un buon punto di partenza per quelli che iniziano a programmare. Vediamo le sue caratteristiche principali:
  • linguaggio interpretato, non compilato;
  • scrittura ordinata del codice;
  • sintassi semplice ed intuitiva.
Le applicazioni di Python sono molteplici: dalla gestione di una marea di dati in tabella, ai grafici di funzioni matematiche... all'analisi di espressioni più o meno semplici, come potete vedere nella pagina... :)

Funzioni built-in di Python

Python offre moltissime funzioni già fatte che possiamo utilizzare per i nostri programmi, senza doverle ogni volta ricreare. Qui una lista delle più utili e comuni:

  • Le funzioni I/O (Input/Output)

    In Python sono due le funzioni più utilizzate nei flussi di I/O: input() e print().

    La prima permette di creare un flusso di dati che consente all'utente di inserire dei dati nel programma. 

    
    input("Inserisci un numero: ")
    >>> Inserisci un numero: _

    NB: i dati inseriti in input vengono sempre inseriti come stringhe, a meno che non si usino funzioni come eval().

    La seconda permette, invece, di visualizzare delle stringhe in console. 

    
    print("ciao", nomeInserito, ":D")
    >>> ciao Christian :D

    NB: si possono inserire più argomenti all'interno di ogni print: verrà stampato ogni argomento, uno dopo l'altro, lasciando un carattere vuoto tra ognuno.

  • Int(), string(), list()...

    In Python esistono molte funzioni che permettono di tramutare una variabile da un tipo ad un altro. Vediamo qualche esempio:

    • int()

      Questa funzione permette di trasformare una variabile stringa o float in un intero. 
      
    stringa = "13"
    print(int(stringa) + 2)
    >>> 15

    • string()

      Questa funzione, invece, trasforma una variabile di qualsiasi tipo in una stringa. 
      
    numeroClasse = 3
    print("Classe:", string(numeroClasse) + "B")
    >>> Classe: 3B

    • list()

      Questa funzione, infine, permette di tramutare in lista una struttura dati in sequenza, come un set o una tupla, per esempio. 
      
    tupla = ("Lunedì", "Martedì", "Mercoledì")
    print(list(tupla))
    >>> list("Lunedì", "Martedì", "Mercoledì")
    Ci sono molte altre funzioni simili a queste, come tuple(), float()... le vedrete programmando all'occorrenza ;)

  • len()

    Questa è una funzione che vi tornerà utile spesso, davvero. Ritorna la quantità di elementi che una sequenza contiene, cioè una qualsiasi struttura dati che contiene delle variabili, in intero. 
    
    lista = [1, 2, 3, 4, 5, 6, 7]
    print(len(lista))
    >>> 7

Definire una funzione

Bene, adesso vediamo come possiamo noi definire una funzione all'occorrenza. Per definire una funzione bisogna anteporre al blocco di codice la keyword def, successivamente, separato con uno spazio, il nome della funzione, le parentesi con gli argomenti richiesti subito dopo, ":" e, andando a capo con un'indentazione, si può dichiarare ciò che deve eseguire la funzione quando richiamata. 


    def funzione(a, b):
        return a + b
    print(funzione(1, 2))
    >>> 3
NB: per richiamare la funzione, questa dev'essere dichiarata nelle righe precedenti.

Funzioni recursive

Abbiamo visto nella sezione precedente la keyword return. Questa, come avrete intuito, dichiara a Python cosa la funzione deve ritornare al resto del codice, in quel caso la somma tra i due argomenti immessi come parametri richiesti nelle parentesi.

Se nel return della funzione viene richiamata la funzione stessa, si parla di funzione "recursive", cioè che la funzione richiama essa stessa all'interno del proprio codice. 


    def countDown(count = 3):
        print(count)
        count -= 1 // diminuisce di 1 ad ogni richiamo
        if count > 0:
            countDown(count)
        else:
            print("Finished!")
    countDown()
    >>> 3
    >>> 2
    >>> 1
    >>> Finished!

Tipi di variabili

In Python ci sono molti tipi di variabili di cui poter usufruire, alcuni li possiamo ritrovare in altri linguaggi di programmazione, altri sono nativi di Python. Per dichiarare una variabile, a differenza degli altri linguaggi, bisogna semplicemente scrivere il nome della variabile (chiamato 'identificatore') e assegnargli un valore.

  1. integer 
    
    a = 2
    altro_int = 5
  2. float 
    
    a = 2.5
    altro_float = 19.003
  3. Nativo di Python: string
    Possiamo usare sia i singoli apici ' ' che i doppi apici " ", e per definire una stringa contenente altri apici bisogna utilizzare o entrambi i tipi (es. i doppi all'esterno e i singoli all'interno) se sono pochi i casi, o altrimenti usare ' \" '. Questo backslash è un carattere che viene molto utilizzato in casi come questi, come per esempio anche nelle RegEx (Regular Expressions, ne parlerò più avanti). 
    
    stringa = 'La stringa "hola" è lunga 4 caratteri'
    print(stringa)
    >>> La stringa "hola" è lunga 4 caratteri

    stringa = "La stringa \"hola\" è lunga 4 caratteri"
    print(stringa)
    >>> La stringa "hola" è lunga 4 caratteri
  4. Nativo di Python: list
    Una lista è un insieme di elementi di qualsiasi tipo, e quando bisogna riferirsene bisogna utilizzare le parentesi quadre all'inizio e alla fine.
    NB: per accedere direttamente ad un singolo elemento della lista bisogna richiamarlo col suo indice, come è scritto nella penultima riga. 
    
    myList = [1, "2", [3, 4]]
    for element in myList:
        print(element)
    >>> 1
    >>> "2"
    >>> [3, 4]
    print(myList[1])
    >>> "2"
  5. Nativo di Python: tuple
    Una tupla è come una lista ma le parentesi sono tonde e i suoi elementi sono immutabili e fissi in numero. 
    
    myTuple = (1, 0.5, "elemento")
    myTuple[2] = 5 // questa riga darà errore
    >>> TypeError: 'tuple' object does not support item assignment
  6. Nativo di Python: dict
    Un dizionario è un insieme di coppie di elementi, dove il primo della quale è la "chiave" e il secondo è il "valore". Per accedere ad un suo elemento, infatti, dobbiamo usare la dicitura 'dizionario[chiave] = valore', dove la chiave è solitamente una stringa che identifica il valore, e il valore può essere di qualsiasi tipo. 
    
    myDict = {"name": "Mario", "surname": "Rossi", "age": 20}
    print(myDict[name], myDict[surname])
    >>> Mario Rossi
Infine, ci sono anche altre strutture dati utilizzabili in Python, grazie alla fiorente community di programmatori che lavorano regolarmente offrendo nuovi moduli e nuove librerie agli utenti che scrivono in questo linguaggio (queue, stack, ...).

Variabili globali e locali

In Python, come in altri linguaggi di programmazione, le variabili hanno uno "scope", cioè possono essere globali o locali.

  • Le variabili globali sono accessibili da ogni parte del codice 
    
    var = 5
    def function():
        print(var) // dichiarata al di fuori della funzione, ma essendo globale sarà accessibile
    function()
    >>> 5
  • Le variabili locali, invece, sono accessibili solo nella funzione o classe in questione. 
    
    def function():
        var = 20
    print(var) // la funzione print non riuscirà a trovare nessuna variabile con quel nome
    >>> NameError: name 'var' is not defined
